Frequently Asked Questions

Find quick answers to common questions about our software, live wallpapers, and optimization tools.

General & Safety

Are BSR Studios applications really 100% free?
Yes! All our core utility applications, including AnyWebWall, AnyVidWall, and Any Screen Recorder, are completely free to download and use. We do not use paywalls, hidden subscriptions, or annoying watermarks on your outputs.
Will this software work on my low-end 2GB RAM PC?
Absolutely. Our software is engineered specifically for low-end hardware. By utilizing native C++, OpenGL, and Direct3D APIs instead of heavy web browsers, we ensure our apps consume near 0% CPU and minimal RAM, preventing game lag.
Why is Windows Defender showing a blue warning screen during installation?
Since we are an independent development studio offering free software, Windows SmartScreen may flag our new executable files (.exe) as unrecognized. This is a common 'False Positive'. Our software is 100% safe, free of malware and telemetry. Simply click 'More info' and then 'Run anyway' to install it safely.
Are there any ads inside the desktop applications?
No. While our website may contain advertisements to support our server costs, our downloadable Windows desktop applications are 100% clean and ad-free.

AnyWebWall & .aww Wallpapers

What is an .aww file and how do I open it?
An .aww file is our proprietary interactive wallpaper format. Unlike standard videos, it contains C++ and OpenGL logic for real-time 3D rendering and mouse interaction. You can only open and apply these files using our AnyWebWall Engine.
Can I set my own MP4 video using AnyWebWall?
No. AnyWebWall is specifically designed to run interactive 2D/3D .aww scripts. If you want to set standard video files (MP4, MKV, AVI) as your desktop background, please download our dedicated video wallpaper tool: AnyVidWall.
Why is my AnyWebWall screen black?
If the engine is running but your screen is black, it usually means a file failed to load properly. Open your Windows Task Manager (Ctrl+Shift+Esc), end the 'AnyWebWall' task, and restart the application.

Any Screen Recorder

Why are the raw (.screc) files so large during recording?
To achieve our signature 0% CPU usage during your gameplay, Any Screen Recorder captures uncompressed raw data (the .screc files) instead of encoding and compressing the video in real-time. This saves your PC's performance. Once you finish playing and close the game, you can click 'Turn into Video' in the app to compile this large raw file into a highly compressed, standard-sized MP4 video.
Can I record my webcam (Facecam) with this tool?
Currently, Any Screen Recorder focuses strictly on capturing high-performance screen data and internal system audio. Facecam overlay is not supported in the current version.
Is there a time limit for my screen recordings?
No, there are zero time limits. You can record for hours, provided you have enough free storage space on your hard drive (preferably an SSD) to store the temporary raw data files.
